# Assets folder

Place your icon files here:

- `icon.png` - 256x256 PNG (required for development)
- `icon.ico` - Windows icon (required for Windows build)
- `icon.icns` - macOS icon (optional)

## Generate icons

1. Create a 512x512 or 1024x1024 PNG logo
2. Use an online converter:
   - https://icoconvert.com/ (for .ico)
   - https://cloudconvert.com/ (for .icns)

## Placeholder

Until you have a real icon, you can use any 256x256 PNG image.
